ChainProof "Trustless bug bounties — exploits verified on-chain, rewards paid automatically."

TrustBounty — Decentralized Zero-Trust Bug Bounty Platform A full-stack Web3 project where companies lock ETH rewards into a Solidity smart contract on Ethereum Sepolia, hackers submit AES-256 encrypted exploit reports via IPFS, and an autonomous oracle verifies exploits inside an isolated Docker sandbox before triggering automatic on-chain payment.
